Pickle - модуль, используемый для сериализации и десериализации объектов python. В качестве таких объектов могут выступать списки, словари, кортежи, а также объкты классов, созданных разработичком.
Picle используется для хранения объектов в Python.

+ Нам не нужно создавать один и тот же объект снова и снова. Создаем объект один раз, сохраняем его на диск, а позже мы загружаем этот объект с диска, не создавая объект вновь.
+ Возможность сериализовать практически любой объект Python без какого-либо шаблона.
+ Возможность сохранения скрипта в определенном состоянии - сбросить состояние программы в данный момент и загрузить скрипт позже с сохраненным состоянием.

- Характерен только для Python, следовательно модуль не гарантирует межъязыковой совместимости.
- Разные версии Python несовместимы друг с другом. Следовательно, использование модуля pickle в версии Python 2.x не гарантирует кррректную загрузку объекта в версии Python 3.x.
- Т.к. сохранять можно любые объекты, а при загрузке объекты не проходят проверку (это лишь набор байтов), то следует избегать извлечения данных из неизвестных источников, поскольку они могут содержать ошибочные данные или вирусы.
- Сохраненный объект не читаем, в отличие от использования JSON формата для объектов.
